# 自分だけの買い物リストを作成しよう

Reactを勉強したので、試しに作ったアプリです。
ユーザー登録すると、買い物リストを使えます。パソコン、スマホから登録できるので、
便利かもしれません？（パソコンをよく使う人は便利かも）

下記ＨＰからアクセスできます。
https://junk-programing.github.io/shopping-memo/

## セットアップ～インストール～スタート
リポジトリをクローンします

git clone https://github.com/Junk-Programing/shopping-mem.git

コマンドにより、pitch-recognitionフォルダ直下で下記のコマンドを使い、必要なモジュールをインストールします。

npm install

下記コマンドで実行できます。

npm start
